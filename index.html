Guia de Instalação

Montagem completa do sistema de contagem de catraca com Arduino, Python e Firebase. O script Python envia o status de conexão do Arduino ao Firebase em tempo real — este monitor exibe alertas quando o cabo é desconectado.

Arduino Uno / Mega Python 3.8+ Firebase RTDB

01
Hardware — Conexão do Arduino

Compatível com Arduino Uno e Mega. Ligação idêntica em ambos.

PinoConectar emDescrição
D2Sinal do sensor / contato secoINPUT_PULLUP — LOW = giro detectado
GNDGND do sensorTerra comum
USBPC / Raspberry PiSerial + alimentação 5V

Teste de bancada: encoste um fio entre D2 e GND para simular uma passagem — LED 13 piscará.

02
Firmware — Código do Arduino

Carregue via Arduino IDE. Debounce de 300 ms evita contagem dupla.

/* Firmware Catraca — Arduino Uno/Mega */ const int PINO_SENSOR = 2; const int PINO_LED = 13; const int DEBOUNCE_MS = 300; bool estado_anterior = HIGH; unsigned long ultimo_giro = 0; void setup() { Serial.begin(9600); pinMode(PINO_SENSOR, INPUT_PULLUP); pinMode(PINO_LED, OUTPUT); delay(1000); Serial.println("PRONTO"); } void loop() { bool estado_atual = digitalRead(PINO_SENSOR); unsigned long agora = millis(); if (estado_anterior == HIGH && estado_atual == LOW) { if (agora - ultimo_giro > DEBOUNCE_MS) { ultimo_giro = agora; digitalWrite(PINO_LED, HIGH); Serial.println("GIRO"); delay(100); digitalWrite(PINO_LED, LOW); } } estado_anterior = estado_atual; }
03
Python — Script com status do Arduino

Grava /status/arduino no Firebase em cada mudança: online (conectado), offline (cabo fora) ou erro (acesso negado — feche o Arduino IDE).

import serial, serial.tools.list_ports import time, threading, logging from datetime import datetime import firebase_admin from firebase_admin import credentials, db FIREBASE_DB_URL = "https://SEU-PROJETO-rtdb.firebaseio.com" BAUD_RATE = 9600 # Credenciais embutidas (não vai para o git) FIREBASE_CREDENTIALS = { "type": "service_account", "...": "..." }

05
Executando o sistema
  • Instalar: pip install pyserial firebase-admin
  • Carregar o firmware no Arduino via Arduino IDE
  • Fechar o Arduino IDE (Monitor Serial bloqueia a porta — causa o PermissionError 13)
  • Executar: python monitor_catraca.py
# Windows — rodar como serviço com NSSM nssm install CatracaMonitor "python.exe" "C:\catraca\monitor_catraca.py" nssm start CatracaMonitor # Linux — systemd service sudo nano /etc/systemd/system/catraca.service # ExecStart=/usr/bin/python3 /opt/catraca/monitor_catraca.py sudo systemctl enable --now catraca
